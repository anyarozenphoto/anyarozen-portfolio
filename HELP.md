# 📖 Руководство по управлению сайтом Anya Rozen

Весь сайт управляется через **3 файла**. Ниже — точные инструкции для каждого действия.

---

## 📁 Структура файлов

| Файл | Что делает |
|---|---|
| `src/data/navigation.ts` | Управляет меню, порядком страниц и их видимостью |
| `src/data/photos.ts` | Хранит все ссылки на фотографии |
| `src/pages/About.tsx` | Текст страницы About |

---

## 1. Показать / скрыть страницу

**Файл:** `src/data/navigation.ts`

Каждая страница — это одна строка. Найди нужную и поменяй `hidden`:

```ts
{ to: "/maternity", label: "Maternity", photosKey: "maternity", hidden: true },
```

- **`hidden: true`** → страница НЕ видна в меню и на главной
- **`hidden: false`** → страница видна везде

> ⚠️ Даже скрытая страница доступна по прямой ссылке (например `yoursite.com/maternity`).

---

## 2. Изменить порядок страниц в меню

**Файл:** `src/data/navigation.ts`

Порядок строк = порядок в меню и на главной. Просто перемести строки:

```ts
// Было:
{ to: "/portraits", label: "Portraits", ... },
{ to: "/lifestyle", label: "Lifestyle", ... },

// Стало (Lifestyle теперь выше):
{ to: "/lifestyle", label: "Lifestyle", ... },
{ to: "/portraits", label: "Portraits", ... },
```

---

## 3. Добавить фотографии на страницу

**Файл:** `src/data/photos.ts`

Найди массив нужной страницы и добавь URL картинок:

```ts
maternity: [
  "https://ссылка-на-фото-1.jpg",
  "https://ссылка-на-фото-2.jpg",
],
```

### Правила:
- Каждая ссылка в кавычках `"..."`
- После каждой ссылки запятая `,`
- Порядок ссылок = порядок фото на странице
- Первое фото в массиве = обложка карточки на главной странице

---

## 4. Удалить фотографию

**Файл:** `src/data/photos.ts`

Просто удали строку с URL из нужного массива.

**Было:**
```ts
portraits: [
  "https://фото-1.jpg",
  "https://фото-2.jpg",   ← удалить эту строку
  "https://фото-3.jpg",
],
```

**Стало:**
```ts
portraits: [
  "https://фото-1.jpg",
  "https://фото-3.jpg",
],
```

---

## 5. Изменить порядок фотографий

**Файл:** `src/data/photos.ts`

Поменяй местами строки с URL в нужном массиве. Первая строка = первое фото.

---

## 6. Изменить hero-фото на главной странице

**Файл:** `src/data/photos.ts`

Замени URL в строке `hero`:

```ts
hero: "https://новая-ссылка-на-hero.jpg",
```

---

## 7. Изменить текст на странице About

**Файл:** `src/pages/About.tsx`

Найди блок между комментариями:

```tsx
{/* ✏️ EDIT YOUR BIO TEXT BELOW */}

<p>Твой текст здесь...</p>
<p>Второй абзац...</p>

{/* ✏️ END BIO TEXT */}
```

### Что можно делать:
- Менять текст внутри тегов `<p>...</p>`
- Добавлять новые абзацы: `<p>Новый абзац</p>`
- Удалять абзацы — удали всю строку `<p>...</p>`

---

## 8. Добавить новую страницу-галерею (например "Wedding")

### Шаг 1 — Добавь массив фото
**Файл:** `src/data/photos.ts`

Добавь новый массив перед закрывающей `};`:
```ts
wedding: [
  "https://фото-1.jpg",
  "https://фото-2.jpg",
],
```

### Шаг 2 — Создай файл страницы
Скопируй файл `src/pages/Portraits.tsx`, назови `src/pages/Wedding.tsx`.
Внутри замени:
```ts
// Было:
const images = photos.portraits;
// Стало:
const images = photos.wedding;
```

### Шаг 3 — Добавь маршрут
**Файл:** `src/App.tsx`

Добавь import вверху:
```ts
import Wedding from "./pages/Wedding";
```

Добавь Route внутри `<Routes>`:
```tsx
<Route path="/wedding" element={<Wedding />} />
```

### Шаг 4 — Добавь в навигацию
**Файл:** `src/data/navigation.ts`

Добавь строку в нужное место (порядок = позиция в меню):
```ts
{ to: "/wedding", label: "Wedding", photosKey: "wedding", hidden: false },
```

> Также добавь `"wedding"` в тип `photosKey` в интерфейсе `NavItem` в том же файле.

## Краткая шпаргалка

| Хочу... | Файл | Что делать |
|---|---|---|
| Показать/скрыть страницу | `navigation.ts` | `hidden: true/false` |
| Поменять порядок в меню | `navigation.ts` | Переставить строки |
| Добавить/удалить фото | `photos.ts` | Добавить/удалить URL |
| Поменять hero на главной | `photos.ts` | Заменить URL в `hero` |
| Изменить текст About | `About.tsx` | Редактировать `<p>` теги |
| Новая страница | Все 4 файла | См. раздел 8 |
